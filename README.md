# Задание 1
Написать функцию, которая рекурсивно обходит указанную директорию и возвращает список всех файлов с их размерами и правами доступа.

## Функционал
- Рекурсивно обходит все поддиректории указанной директории
- Для каждого файла собирает следующую информацию:
  - Полный путь к файлу
  - Размер файла в байтах
  - Права доступа к файлу (в восьмеричном формате)
- Обрабатывает исключения PermissionError и FileNotFoundError для повышения надёжности
- Выводит список всех найденных файлов в удобном формате

## Использование
```bash
python lab1.py
```

После запуска программа запросит ввести путь к папке для сканирования.
